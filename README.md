## 📌 Features

- 🎨 Interactive 3D environment for interior design  
- 🛋️ Dynamic object rendering (sofa, decor items, etc.)  
- 🔄 Real-time customization of textures and styles  
- ⚡ Smooth UI with responsive design  
- 🌐 Full-stack architecture (MERN)  

---

## 🛠️ Tech Stack

### Frontend
- React.js  
- Three.js / React Three Fiber  
- Tailwind CSS  

### Backend
- Node.js  
- Express.js  

### Database
- MongoDB

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository
```bash
git clone https://github.com/aaqeebpinjari/Ambiance-Adornment.git
cd Ambiance-Adornment 

```

2️⃣ Install dependencies

Frontend
```bash
cd client
npm install

```
Backend
```bash
cd server
npm install
```

3️⃣ Run the project

Start backend
```bash
npm run dev

```

Start frontend
```bash
npm start
```

🔐 Environment Variables
Create a .env file in the server folder:
```bash
MONGO_URI=your_mongodb_connection
PORT=5000
```
